# Java Learning Games

A collection of small console games built while learning Java.

This project is used to practice core programming concepts such as:
- classes and methods
- input validation
- loops and game logic
- random number generation
- clean program structure

---

## Games

### Guessing Game
A number guessing game where the player must guess a random number between 1 and 100 within a limited number of attempts.

Features:
- input validation
- feedback for high / low guesses
- invalid input handling

More games will be added as I continue learning Java.
